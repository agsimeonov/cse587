To run the code and set up the environment:
1. Place .xls files in a directory named "data_set" 
   residing in the same directory as the .r file.
2. Open the .r file in R or R Studio.
3. Run source("<path to file>/<filename>.r")
4. Wait for everything to initialize.
5. You should now be able to interact with and run the code.
